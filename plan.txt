Написать приложение которое будет осуществлять создание/редактирование списка населенных пунктов и их характеристика.
Входящие данные:
- Названия на разных языках (русский, украинский и английский), в будущем языков может быть больше;
- количество населения;
- координаты центра населенного пункта (широта, долгота);
- Источник с данными о температуре воздуха для этого пункта, один источник может быть использован для нескольких населенных пунктов;
- Координаты источника(широта, долгота);
- Название источника(текст) ;
- Температура.

В будущем планируется по этим данным делать выборку по населению или осуществлять поиск по названию, предусмотреть эти возможности при проектировании структуры бд. Реализовывать на PHP поиск по названию и по населению не нужно, только в БД.

При добавлении населенного пункта нужно назначить источник(комбобокс), в вариантах комбобокса источников присутствует пункт "Ближайший" который назначается автоматически при сохранении связь населенного пункта с ближайшим источником.
При редактировании в комбобоксе "источник" данные сортируется по удаленности от пункта в радиусе 100 километров.
